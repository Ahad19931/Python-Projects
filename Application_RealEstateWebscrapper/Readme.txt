Real State Webscrapper.

This application allows the user to scrap the web data of a real state website and save it in a .csv file.

1. Run main.py in order to run the application.

Following dependencies need to downloaded before running the code:
	1) Pandas
	2) Beautiful Soup
	3) Requests

Note: This main.py code will only work for the following url:

http://www.pyclass.com/real-estate/rock-springs-wy/LCWYROCKSPRINGS/

To use it for any other url, changes need to be done in the .py file.
